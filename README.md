# s21_matrix

Собственная библиотека для работы с матрицами на языке **C**.

Проект реализован в виде статической библиотеки `s21_matrix.a` и включает базовые операции линейной алгебры: создание и удаление матриц, сравнение, арифметические операции, транспонирование, вычисление определителя, матрицы алебраических дополнений и обратной матрицы.

---

## О проекте

В рамках проекта была реализована собственная библиотека для обработки числовых матриц без использования сторонних решений.  
Основная цель - вручную написать базовые матричные операции, аккуратно организовать работу с динамической памятью и покрыть функциональность unit-тестами.

---

## Возможности библиотеки

### Создание и удаление матриц
- `s21_create_matrix` - создание матрицы заданного размера
- `s21_remove_matrix` - освобождение памяти и очистка матрицы

### Сравнение матриц
- `s21_eq_matrix` - проверка матриц на равенство с учетом точности

### Арифметические операции
- `s21_sum_matrix` - сложение матриц
- `s21_sub_matrix` - вычитание матриц
- `s21_mult_number` - умножение матрицы на число
- `s21_mult_matrix` - умножение матрицы на матрицу

### Преобразования
- `s21_transpose` - транспонирование матрицы

### Вычислительные операции
- `s21_determinant` - вычисление определителя
- `s21_calc_complements` - вычисление матрицы алгебраических дополнений
- `s21_inverse_matrix` - нахождение обратной матрицы

## Структура матрицы

```c
typedef struct matrix_struct {
    double **matrix;
    int rows;
    int columns;
} matrix_t;
```

---

## Коды возврата

Большинство функций библиотеки возвращают:

- `0` — операция выполнена успешно
- `1` — некорректная матрица
- `2` — ошибка вычисления

Для сравнения матриц используются отдельные значения:

```c
#define SUCCESS 1
#define FAILURE 0
```

## Пример использования

```c
#include "s21_matrix.h"

int main() {
    matrix_t A, B, R;

    s21_create_matrix(2, 2, &A);
    s21_create_matrix(2, 2, &B);

    A.matrix[0][0] = 1;
    A.matrix[0][1] = 2;
    A.matrix[1][0] = 3;
    A.matrix[1][1] = 4;

    B.matrix[0][0] = 5;
    B.matrix[0][1] = 6;
    B.matrix[1][0] = 7;
    B.matrix[1][1] = 8;

    if (s21_sum_matrix(&A, &B, &R) == 0) {
        s21_remove_matrix(&R);
    }

    s21_remove_matrix(&A);
    s21_remove_matrix(&B);

    return 0;
}
```
